# powershell-lsp

PowerShell language server (PowerShell Editor Services) for Claude Code, providing type checking and code intelligence.

## Supported Extensions

`.ps1`, `.psm1`, `.psd1`

## Prerequisites

- PowerShell 7+ (`pwsh`)
- PowerShellEditorServices module (install via PowerShell Gallery):

```powershell
Install-Module PowerShellEditorServices -Scope CurrentUser -Force
```
